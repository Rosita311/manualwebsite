Introductie

Ik wil heel graag aan het werk als (web)developer. Ik vind het namelijk erg leuk om websites te bouwen. Momenteel volg ik een Werk-Fit traject via het UWV om uit te zoeken, waar mijn interesses liggen (deze heb ik nu dus gevonden), daarnaast weet ik nog niet hoeveel uur ik kan werken en wat voor werkplek bij mij past. Ook is het belangrijk om een goede blalans te vinden tussen werk en privé. Naast het Werk-Fit traject volg ik een online cursus webdevelopment bij Codecademy en heb ik sinds september 2016 mijn eigen blog op rulesbyrosita.nl

Cerebrale Parese

Ik heb Cerebrale Parese(CP). CP is een houdings- en bewegingsstoornis als gevolg van een hersenbeschadiging die plaats vond voor de eerste verjaardag van het kind. CP heeft verschillende oorzaken, zolang het maar plaats vond rond de geboorte en in het eerste levensjaar. Loopt iemand na het eerste levensjaar een hersenbeschadiging op, dan is er sprake van Niet Aangeboren Hersenletsel(NAH). Cerebrale Parese valt ook onder hersenletsel.Hier lees je meer over CP.

Cerebrale Parese bij mij

Ik heb cerebrale parese gekregen doordat ik 3 maanden te vroeg ben geboren en ik daarbij een zuurstofgebrek heb gehad, hierdoor heb ik een hersenbeschadiging opgelopen en ben ik spastisch aan de rechterkant van mijn lichaam.

Tot mijn 33ste heb ik altijd gedacht dat ik alleen maar fysieke beperkingen had als gevolg van mijn CP. Dat bleek dus niet zo te zijn. Vanaf 2010 t/m 2017 studeerde ik Maatschappelijk Werk en Dienstverlening op het HBO. Dat ging goed tot ik helemaal vastliep tijdens mijn derdejaars stage. Omdat ik tegen een burn-out aanliep ben ik gestopt. Omdat ik allerlei dingen niet kon en daar ook geen verbetering kwam ben ik op advies van mijn revalidatiearts gaan revalideren in de Sint Maartenskliniek. Ik kreeg daar een Neuropsychologisch onderzoek. Uit dat onderzoek is gebleken dat ik een trage informatieverwerking heb. Daarna heb ik een aantal dingen geleerd, zoals plannen, koken en blindtypen met een hand.

Beperkingen als gevolg van CP

De fysieke gevolgen van mijn handicap zijn: Ik ben slecht ter been: ik kan niet zo goed lopen, ik draag spalken, een traplift in huis en maak gebruik van een driewielfiets. Daarnaast ben ik snel moe en moet ik overdag regelmatig even slapen.

Door de cerebrale parese ben ik niet alleen spastisch aan de rechterkant van mijn lichaam, maar heb ik ook cognitieve beperkingen, dit komt omdat ik een trage informatieverwerking heb. Hierdoor ben ik snel overprikkeld, ervaar ik veel chaos is plannen en organiseren moeilijk, ik heb geen overzicht over verschillende zaken en kan ik maar een ding tegelijk. Daarnaast heb ik moeite met het onder controle houden van mijn emoties. Vanwege de trage informatieververwerking is mijn IQ niet betrouwbaar achterwege gelaten. Praktische voorbeelden zijn: Dat ik mijn spullen regelmatig kwijt ben of vergeet en dat ik vuurwerk of onnodig toeteren vreselijk vind.

Mijn Kwaliteiten

Naast mijn beperkingen, heb ik natuurlijk ook zo mijn kwaliteiten. Dit zijn:

  • Sociaal: Ik ben een kletskous.
  • Ik werk veel achter de computer. Ik beheer mijn eigen website en doe alles zelf (WordPress CMS), ik heb de harde schijf van een laptop vervangen en ik volg een cursus webdevelopment.
  • Ik ben goed in schrijven en taal (Ik zie alle fouten, behalve die van mezelf).
  • Nieuwsgierig – html, css en nu bezig met Javascript
  • Doorzetter - heb hbo gestudeerd en heb mijn zoon een tijd alleen opgevoed.

Wat ik belangrijk vind

  • Ik vind het belangrijk om zelfstandig te kunnen zijn en ik probeer ook zoveel mogelijk zelf te doen.
  • Ook vind ik het belangrijk om betrokken te zijn bij mensen om me heen en bij mijn (vrijwilligers)werk.
  • Ik vind openheid en transparantie belangrijk
  • Ik vind het belangrijk om goed te kunnen samenwerken.
  • Betrouwbaarheid vind ik belangrijk.

Ik denk dat de waarden die ik hierboven heb aangegeven erg belangrijk zijn in het werk van een webontwikkelaar. Deze waarden komen in elk werkveld wel van toepassing.

Over deze website

Ik maak deze website omdat ik graag aan het werk wil als (web)developer. Dat betekent naast veel leren dat ik ook veel moet oefenen met het daadwerkelijk maken van een website. Toevallig heb ik als opdracht voor De Realisten Academie en FLOOR Jongerencoaching om een gebruiksaanwijzing over mezelf te maken. Nu wil ik beiden combineren, dus bouw ik een website met daarop mijn gebruiksaanwijzing. Voor deze website maak ik gebruik van HTML, CSS en omdat ik nu begin met Javascript te leren, heb ik er een beetje Javascript in gezet. Het doel is om alles wat ik tot nu toe heb geleerd en de dingen die ik bij de Realisten Academie in deze website te stoppen.
