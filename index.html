Introductie

Ik wil heel graag aan het werk als (web)developer. Ik vind het namelijk erg leuk om websites te bouwen. Momenteel volg ik een Werk-Fit traject via het UWV om uit te zoeken, waar mijn interesses liggen (deze heb ik nu dus gevonden), daarnaast weet ik nog niet hoeveel uur ik kan werken en wat voor werkplek bij mij past. Ook is het belangrijk om een goede

Ik heb Cerebrale Parese(CP). CP is een houdings- en bewegingsstoornis als gevolg van een hersenbeschadiging die plaats vond voor de eerste verjaardag van het kind.

Ik maak deze website omdat ik graag aan het werk wil als (web)developer. Dat betekent naast veel leren dat ik ook veel moet oefenen met het daadwerkelijk maken van een website. Toevallig heb ik als opdracht voor De Realisten Academie en FLOOR Jongerencoaching om een gebruiksaanwijzing over mezelf te maken. Nu wil ik beiden combineren, dus bouw ik een website met daarop mijn gebruiksaanwijzing. Voor deze website maak ik gebruik van HTML, CSS en omdat ik nu begin met Javascript te leren, heb ik er een beetje Javascript in gezet. Het doel is om alles wat ik tot nu toe heb geleerd en de dingen die ik bij de Realisten Academie in deze website te stoppen.
